## ✨ Возможности

- 🔐 **Безопасная авторизация** через Discord токен (хранится только в вашем браузере)
- 📁 **Клонирование структуры** сервера:
  - Название сервера
  - Иконка сервера (аватарка)
  - Все роли с правами
  - Категории и каналы
  - Права доступа (overwrites)
- 🧹 **Полная очистка** целевого сервера перед клонированием
- 📊 **Детальные логи** процесса с цветовой индикацией
- 🎨 **Красивый интерфейс** с анимациями и темной/светлой темой
- 📱 **Адаптивный дизайн** — работает на ПК, планшетах и телефонах
- 🔔 **Уведомления** о ходе выполнения (при установке PWA)
- ⚡ **Фоновая работа** через Service Worker

## 🚀 Как использовать

### 1. Получите токен Discord

1. Откройте **Discord в браузере** (web версию)
2. Нажмите `F12` → вкладка **Network**
3. В поле фильтра введите: `/api`
4. Найдите запрос с названием `science`
5. В заголовках запроса найдите поле `authorization` — это ваш токен
6. Скопируйте его (начинается с `MT` или `mfa`)

> ⚠️ **ВАЖНО:** Токен — это ваш пароль. Никому его не показывайте!

### 2. Войдите в приложение

- Вставьте токен в поле на главном экране
- Нажмите «Проверить токен» для подтверждения
- Нажмите «Войти»

### 3. Выберите серверы

- **Исходный сервер** — откуда копировать структуру
- **Целевой сервер** — куда копировать (все существующие каналы и роли будут удалены!)

### 4. Начните клонирование

- Нажмите «Начать клонирование»
- Подтвердите правила использования
- Ожидайте завершения процесса

## 🛠 Установка

### Локальный запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/xolirx/DiscordCloner.git

# Перейдите в папку проекта
cd DiscordCloner

# Откройте index.html в браузере
# Или используйте любой локальный сервер, например:
npx serve .
```

### Деплой на GitHub Pages

1. Форкните репозиторий
2. В настройках репозитория включите **GitHub Pages**
3. Выберите ветку `main` и папку `/ (root)`
4. Сайт будет доступен по адресу: `https://[ваш-username].github.io/DiscordCloner`
